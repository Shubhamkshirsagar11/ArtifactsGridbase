Developer notes
Mapping — behavior spec
Prototype annotation — not part of the product UI
CONTINUE GATE
Continue is disabled until zero classes are Unmapped. Not Available counts as handled — it is a deliberate human choice ("we have no role like this"), and nothing ever assigns it automatically.
PICK-ONLY ROLES
The role dropdown is pick-only: roles are a curated company list; new roles are added in Settings, never invented here.
PER-ROW SUGGESTIONS
The blue "↳ Use …" link fills that one row's dropdown when clicked — per-row suggestions only; there is no bulk auto-apply. (The Auto-suggest button was removed.)
SAME NAME, TWO BOOKS
The same class name may appear in two rate books and map independently — consumers always know which book they're in, so the right wage resolves.
SINGLE RATE BOOK
With a single rate book, no book-header rows appear — the table looks exactly like the current screenshot minus the Auto-suggest button.
